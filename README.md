# Online-Questionnaire

## 项目介绍

JavaWeb实验作业，基于JSP实现的简单的在线问卷调查系统，使用了MVC架构，实现了用户注册、登录、回答问卷、设计问卷、分析问卷的功能。

## 开发环境

- 操作系统：macOS 10.15.5
- JDK：Java SE 13
- IDE：IntelliJ IDEA Version 2019.2.4 (IU-192.7142.36)(如果需要在Eclipse运行该项目可能需要不同的配置手段)
- 数据库：MySQL 8.0.19
- 服务器：Tomcat 8.5.51.0

M:Model
V:View,jsp
C:Controller,Servlet
